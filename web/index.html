gonnect-vpn-example WASM Demo

Overview

Gonnect is a networking go library providing some helpers and common interfaces that forms a core of gonnect ecosystem.
This project demonstrates how to build a simple p2p VPN using libs from gonnect ecosystem.
This page is a WASM based web client for this VPN build around userspace TCP/IP stack by gVisor wrapped into vtun by gonnect-netstack lib.

Instruction

  • You can start cli VPN server from this project repo with just serve.
  • Request a target URL that is reachable inside the VPN network, for example http://10.200.1.3/
